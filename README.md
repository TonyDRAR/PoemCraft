## Installation

Prerequisite: Python 3.10 or newer.

On Windows PowerShell:

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
python main.py
```

If `py` is not available, use the full path to Python or `python` instead.

On macOS/Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt
python main.py
```

On Linux, if the app fails to open because `tkinter` is missing, install it with your package manager:

```bash
sudo apt install python3-tk
```
